## 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of laptop sales performance using interactive visualizations and KPI tracking. The dashboard helps stakeholders understand sales trends, product performance, and business growth through data-driven insights.

---

## 🎯 Objectives

- Analyze overall laptop sales performance
- Track revenue and transaction metrics
- Monitor sales trends and business KPIs
- Identify top-performing products
- Support strategic decision-making using data

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling
- Microsoft Excel
- Data Visualization

## 🚀 How to Use

1. Download or clone the repository.
2. Open `Project 1.pbix` in Power BI Desktop.
3. Refresh the dataset if required.
4. Explore the dashboard and interact with filters and visualizations.
